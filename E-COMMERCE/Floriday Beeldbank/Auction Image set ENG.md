## Situation:
The link to the old RFH Image Bank will expire on 31 May 2026.  
Photos can no longer be downloaded via the old image bank.

## Solution:  
In your system, go to Constants → Auctions → EKT → External photo URL for Photo ID – EKT import.  
Enter the following in Prefix for the PhotoID: https://image.floriday.io/beeldbank/  
Enter the following in Suffix after the PhotoID: .jpg

If the old Aalsmeer image bank was used at the auction, it does not need to be removed and can remain set up at the auction.
