# Florisoft Inventory App Manual

In this manual, you can read about the use and configuration of the Florisoft Inventory App. With this app, you manage your stock and keep it synchronized with the actual stock locations in your warehouse.

The app consists of the following parts:

## Stock Counting

With the Stock Counting app, you check stock and make sure stock levels remain synchronized with the data in Florisoft.

- [Policy manual](https://github.com/florisoft/User.Manuals/blob/main/CLOUD%20APPLICATIONS/Apps%20Android/App%20Inventory/Stock%20Counting/Policies%20Stock%20Counting%20EN.md)  
- [Quickstart manual](https://github.com/florisoft/User.Manuals/blob/main/CLOUD%20APPLICATIONS/Apps%20Android/App%20Inventory/Stock%20Counting/Quickstart%20Manual%20EN.md)  

## Slotting

The Slotting app is an application for warehouse management that allows products to be linked to locations.

- [Full manual and policies (EN)](Slotting/Slotting%20Manual%20-%20EN.md)

## Consolidation

The Consolidation function is a simple, shared workflow used by both **Slotting** and **Stock Counting**. It helps warehouse employees quickly link parcels to locations, so stock is correct and follow-up actions can be carried out easily.

Short step overview:

- Scan the target location
- Scan or select one or more parcels
- Confirm the link

Additional actions you can easily perform, if needed: check and correct stock, report a quality issue, or print price or stock labels. Consolidation is intended as a practical, user-focused action; the technical settings and policies are managed centrally in the Backoffice and usually do not need to be changed by users.

- [Full manual and policies (EN)](Consolidation/Consolidation%20Manual%20-%20EN.md)
